404s to Mars Hackaton UDIT 2025

ABSTRACT
Desarrollo de una página de error 404 incluyendo elementos semánticos, técnicas de storytelling, responsividad  coherencia estética y de código con el concepto "Aterrizaje en Marte fallido"

INSPIRACIÓN
A través de el análisis de la consola de comandos de los videojuegos franquicia Fallout por parte de une de les desarrolladores (Selina) se desarrolla una pantalla que recuerda a los principios de internet y el concepto del imaginario colectivo de los lenguajes de programación; conectando así con quien visite la página independientemente de su nivel de conocimiento de código.

DESARROLLO
El tiempo de ralización de este proyecto ronda las 6 horas.
Algunos de los retos encontrados han sido la sincronización de los elementos en diferentes tamaños necesarios para implementar una responsividad adecuada, pasando por el proceso habitual de tener el código funcionando a la perfección y en el momento de añadir elementos nuevos de errores y el factor de que el tiempo para desarrollar el proyecto haya sido bastante limitado aunque bastante  más de lo habitual en una Hackaton.
